# Release Notes - v1.1.1

## Resumen
Esta versión menor añade flexibilidad al script de subida a S3, permitiendo seleccionar subcarpetas específicas para carruseles de cumpleaños, y organiza los recursos de imágenes cronológicamente con metadatos de edad.

## Cambios Principales
- **Subcarpetas en Upload CLI:** Se agregaron opciones en `upload.js` para destinos de carruseles de cumpleaños:
  - `carousel Javier Angel` -> `sanahujadev/birthday/ja-sn-rr/carousel`
  - `carousel José Manuel` -> `sanahujadev/birthday/jm-sn-rr/carousel`
